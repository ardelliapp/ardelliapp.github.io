FORM: TEXT BOX

PENDAHULUAN

Form dalam desain web merupakan hal vital yang mutlak harus ada untuk membuat keinteraktifan dengan pengunjung. Dengan adanya form, anda dapat membuat aplikasi buku tamu, online shopping, dll.
Form harus diintegrasikan dengan pemrograman web, misalnya javascript, PHP, ASP, Perl, dll.
Hal ini harus dilakukan karena input yang dimasukkan pengunjung situs ke dalam form harus diolah terlebih dahulu oleh program selanjutnya akan ditampilkan sebagai output dalam browser.

Text box digunakan untuk memasukkan data seperti form yang nantinya juga diisikan data-data dengan tipe data seperti text, password, submit, dll.

TAMPILAN

Masukan data-data yang terkait.

Nama Lengkap:

Username:

Password:


Jika anda submit melalu tombol submit maka data akan ditampilkan pada halaman form.html

ANALISIS

Tampilan seperti yang terlampir. Input digunakan untuk memasukkan data-data, type adalah tipe data yang ingin dimasukkan contoh diatas seperti text, submit, dan password.
